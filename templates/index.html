Maze pathfinder

This program find the shortest path from the start to the arrival of a maze. Click on a wall to make a path and vice-versa. Click on the start cell to move it (re-click somewhere else - except on arrival - to put it) and do the same way with the arrival cell. You can use the buttons on the left to add rows or columns, to empty the maze (only pathes), to fill it (only walls) or to rotate it (180Â°). When you're maze is ready, click on the solve maze button to display the shortest path. Use the reset button if you want to clean your maze (initial number of rows and columns and only walls).
